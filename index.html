About

Guess That Drawing is an idea that was hashed out by our team members for the 371 Software Development Spring 2023 course project. The project is a multiplayer game where one person draws an object while the other players guess what the object is and submit their guesses in a chat box. The project was developed entirely in Unity and git was used for version control.

****Notice of Future Work****

To keep in the integrity of our documentation this notice is posted for all the design elements and features that we did not have time to implement.

  • Each client relies on a local machine running the Unity project and is not currently hosted online or on simmer.io.
  • There is currently no reporting mechanism, or a way to kick players.
  • There are currently no Leaderboards or Rankings
