User Stories

1: Join or host a game

As a player, I want to be able to access an online lobby where I can either join an existing game or create and host my own game.

Elaboration:

The lobby should provide players with the ability to view available games and select a game to join, or create and host their own game. The host should be able to set the game parameters, such as the number of rounds, drawing time, and maximum number of players.

Constraints:

The lobby should be easily accessible and user-friendly, and the game parameters should be clear and easy to adjust.

Effort estimation:

10 person-hours

Acceptance Test:

After creating or joining a game, the player should be able to access the drawing board and chatbox.

2: Access to a drawing board

As a player, I want to have access to a drawing board with a pen and eraser tool to create drawings for players to guess.

Elaboration:

The player should be able to choose from a list of drawing prompts, and then use the pen and eraser tools to create a drawing for the players to guess. The player should also be able to see the players' guesses in the chat box.

Constraints:

The drawing board should be user-friendly, and the prompts should be appropriate for all ages.

Effort estimation:

15 person-hours

Acceptance Test:

After creating a drawing, the player should be able to see the players' guesses in the chat box and the game should move on to the next round after the time limit is up.

3: Guess the drawing and chat with other players

As a player, I want to be able to guess the drawing and chat with other players during gameplay.

Elaboration:

The player should be able to see the drawing on the screen and use the chat box to guess what the drawing is. The game should also keep track of the players' scores, based on the number of correct guesses.

Constraints:

The chat box should be appropriate, and the gameplay should be fast-paced and engaging.

Effort estimation:

20 person-hours

Acceptance Test:

After guessing the drawing correctly, the player should receive points and the game should move onto the next round. The chat box should be appropriate and easy to use.

4: Access to a word bank

As a player, I want to have access to a word bank of drawing prompts to choose from during gameplay.

Elaboration:

The game should provide some list of words for the player to choose from when creating a drawing. The word bank should be diverse and appropriate for all ages.

Constraints:

The word bank should be easily accessible and the words should be appropriate for all ages.

Effort estimation:

5 person-hours

Acceptance Test:

After creating a drawing, the player should be able to choose from a list of words in the word bank.

1.1: Host a server with their friends

As a host, I want to be able to create a private game session and invite my friends to join, so that we can play together and have a fun, private gaming experience.

Elaboration:

The host should be able to create a private game session and invite their friends through a unique link or password-protected access. The game should provide the host with the ability to set the game parameters, such as the number of rounds, drawing time, and maximum number of players.

Constraints:

The private game session should only be accessible to the host and their invited friends, and the game parameters should be clear and easy to adjust.

Effort estimation:

20 person-hours

Acceptance Test:

The host should be able to create a private game session and invite their friends, and the invited friends should be able to access the game session through the unique link or password-protected access. The game parameters should be adjustable by the host, and all players should be able to participate in gameplay.

1.2: Filter games in lobby

As a player, I want to be able to filter available games in the lobby based on criteria such as number of players or game parameters.

Elaboration:

The lobby should allow players to search for and view only the games that meet their desired criteria. Players should also be able to sort the available games based on various parameters such as popularity or time remaining.

Constraints:

The filtering and sorting options should be easy to use and understand.

Effort estimation:

12 person-hours

Acceptance Test:

After filtering and sorting the available games, only games that meet the chosen criteria should be visible in the lobby.

2.3: Choose drawing prompts

As a player, I want to be able to choose a drawing prompt from the word bank to create a drawing for other players to guess.

Elaboration:

The player should be able to browse the word bank for a prompt that inspires them to create a drawing. Once the prompt is chosen, the player can use the pen and eraser tools to create their drawing.

Constraints:

The word bank should be diverse and appropriate for all ages.

Effort estimation:

8 person-hours

Acceptance Test:

After choosing a prompt from the word bank, the player should be able to use the pen and eraser tools to create a drawing.

3.4: Keep track of scores

As a player, I want the game to keep track of my score based on the number of correct guesses I make.

Elaboration:

The game should track the number of correct guesses for each player and update the scores accordingly. The game should also display the scores in real-time for all players to see.

Constraints:

The score-keeping mechanism should be accurate and reliable.

Effort estimation:

10 person-hours

Acceptance Test:

After guessing a drawing correctly, the player's score should increase and the updated scores should be visible to all players.
